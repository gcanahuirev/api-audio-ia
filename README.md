# Simple integration node & python script audio 

## METHOD: /POST
`localhost:3000/api/v1/audio`
`body: voide(file) -> myvoice.wav`

There is a example audio in src/tmp/ directory

## Install dependencies
``` sh
$ yarn
```

## Run server
``` sh
$ yarn start
```

## Environments (opcional)
